# 掷飞刀（Knife Hit）游戏开放文档

## 1. 项目概述

本项目是一个轻量级点击反应类休闲游戏。玩家通过点击屏幕，将飞刀从屏幕下方向上掷出，命中并附着在持续旋转的轮盘上，直到完成目标数量的飞刀命中。

游戏的核心体验是：

- 节奏短平快，单局时长短；
- 操作简单（单击即可）；
- 难点集中在时机判断与节奏控制。

---

## 2. 核心玩法规则

### 2.1 场景布局

1. 屏幕上半部分中间位置放置一个持续转动的轮盘；
2. 屏幕下半部分中间位置放置一把待发飞刀；
3. 背景、UI 和特效可根据美术风格扩展，但不影响核心判定逻辑。

### 2.2 操作方式

1. 玩家点击屏幕；
2. 当前待发飞刀沿竖直方向直线向上移动；
3. 飞刀命中轮盘后，飞刀与轮盘建立附着关系，并跟随轮盘同步旋转；
4. 系统立即在下方生成新的待发飞刀，进入下一次点击准备状态。

### 2.3 胜负规则

1. **胜利条件**：轮盘上成功附着的飞刀数量达到本关指定数量；
2. **失败条件**：飞刀上升过程中与轮盘上已附着的任意飞刀发生碰撞；
3. 胜利或失败后，进入对应结算界面，显示“继续游戏”按钮；
4. 点击“继续游戏”后，开始新一局（重置轮盘、飞刀、计数与状态）。

---

## 3. 一局游戏完整流程

1. 进入关卡，初始化游戏参数（目标飞刀数、轮盘转速、当前命中数）；
2. 轮盘开始旋转；
3. 显示一把待发飞刀；
4. 玩家点击后，飞刀向上飞行；
5. 进行碰撞检测：
   - 若与已附着飞刀碰撞：本局失败；
   - 若成功命中轮盘空隙：附着并计数 +1；
6. 若计数达到目标值：本局胜利；否则继续生成下一把飞刀；
7. 在结算界面点击“继续游戏”，回到步骤 1。

---

## 4. 关键系统设计说明

### 4.1 轮盘系统

- 轮盘保持持续旋转（可配置顺时针/逆时针、转速）；
- 每局开始时重置角度，避免状态残留；
- 可在后续版本扩展为变速、阶段性加速、随机转向等机制。

### 4.2 飞刀系统

- 待发飞刀在底部固定出生点生成；
- 发射后沿世界坐标 Y 轴正方向直线运动；
- 命中轮盘时转换为“已附着飞刀”，其位置与角度由轮盘驱动更新；
- 每次成功命中后，立即补充新的待发飞刀。

### 4.3 碰撞判定系统

- 判定对象：
  - 发射中的飞刀（动态）；
  - 已附着在轮盘上的飞刀（静态相对轮盘、动态相对世界）；
- 判定优先级：
  1. 先判断是否撞到已附着飞刀（失败）；
  2. 否则判断是否到达轮盘附着半径（成功附着）；
- 推荐使用统一的碰撞半径/包围盒策略，确保判定稳定、可调试。

### 4.4 游戏状态机

建议至少包含以下状态：

- `Idle`：等待玩家点击；
- `KnifeFlying`：飞刀发射中；
- `RoundWin`：本局胜利结算；
- `RoundFail`：本局失败结算；
- `RoundReset`：点击“继续游戏”后的重置阶段。

通过状态机控制输入开关与流程推进，避免同一帧重复触发发射或重复结算。

### 4.5 UI 与交互

- 常驻信息：
  - 当前命中数；
  - 目标飞刀数；
  - 剩余飞刀数（可选）；
- 结算界面：
  - 胜利提示/失败提示；
  - “继续游戏”按钮（统一入口，开始新局）；
- 输入限制：
  - 在 `KnifeFlying` / 结算状态下屏蔽重复点击。

---

## 5. 关键参数（建议可配置）

- `targetKnifeCount`：本局需要命中的飞刀数量；
- `wheelRotateSpeed`：轮盘基础旋转速度；
- `knifeFlySpeed`：飞刀上升速度；
- `knifeCollisionRadius`：飞刀碰撞半径；
- `wheelRadius`：轮盘半径；
- `spawnPoint`：飞刀出生点（屏幕下半部中间）。

将以上参数抽离为配置，便于后续关卡化与难度曲线调整。

## 6. 开发验收清单

- [ ] 初始界面中轮盘位于屏幕上半部中间并持续旋转；
- [ ] 初始界面中待发飞刀位于屏幕下半部中间；
- [ ] 点击屏幕后飞刀沿直线向上移动；
- [ ] 飞刀命中轮盘后附着并随轮盘旋转；
- [ ] 成功附着后可继续生成下一把飞刀；
- [ ] 达到指定飞刀数量后触发胜利；
- [ ] 飞刀碰撞到已附着飞刀后触发失败；
- [ ] 胜利/失败界面均有“继续游戏”按钮；
- [ ] 点击“继续游戏”后可正确重开新一局。

## 7. 后续可扩展方向（可选）

- 关卡系统（目标飞刀数逐关提升）；
- 轮盘障碍物（禁止命中区域）；
- 连击与评分系统；
- 皮肤与视觉反馈（命中特效、震屏、音效）；
- 广告复活与道具机制（如减速道具）。
